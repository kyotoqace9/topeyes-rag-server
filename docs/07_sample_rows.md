# knowledge_units サンプル行（v0.1）

本ファイルは、Google Sheets（knowledge_units）に入力する際のサンプルを示す。
ヘッダは「英名（和名）」形式（docs/05, docs/06参照）。

## 運用メモ
- 初回は status=draft で入力し、source_refs（根拠）を埋めてから approved に上げる
- 変更は上書き禁止。version を上げて新行を作成する

---

## サンプル：KU-02（縛り回数カウント）

### 推奨値（要点）
- rule_type: RULE
- status: draft（根拠が揃ったら approved）
- version: v1
- course_name: 7回お約束コースLN
- course_alias: 7回受け取りお約束コース
